# @altrugenix/progress-bar

A premium, accessible, and high-performance React Progress Bar component.

## Installation

```bash
yarn add @altrugenix/progress-bar
```

## Features

- 💎 **Premium Aesthetic**: Modern design with glassmorphism and smooth animations.
- ♿ **Accessible**: Built with accessibility in mind.
- 🧩 **Polymorphic**: Supports the `as` prop for ultimate flexibility (where applicable).
